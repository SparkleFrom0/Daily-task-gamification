# Daily Task Gamification Project



A simple C program that gamifies daily tasks with an XP and energy system.



## Features

- Add new tasks 

- Complete your tasks and gain XP

- Level up when enough XP is collected

- Daily energy refill system

- Save and load progress from file



## How to Compile

On Windows (using GCC / MinGW):

CMD , git-bash or PowerShell;

gcc main.c -o taskgame.exe

taskgame.exe



On Linux/MacOS

gcc main.c -o taskgame

./taskgame
